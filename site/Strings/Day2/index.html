    üìÖ Strings ‚Äì Day 2 - Hashing Pattern

    In this day, we learn how to use hashing (frequency counting) to solve string problems efficiently.

    ‚úÖ Problem 1: Valid Anagram

    üìå Problem Statement

    Given two strings s and t, determine whether t is an anagram of s.

    An anagram means both strings contain the same characters with the same frequency.

    üí° Approach

    Use a frequency counter (hash map).

    If both strings have identical character counts ‚Üí they are anagrams.

    üß† Detailed Explanation

    1. If lengths of both strings are different ‚Üí return False.
    2. Create a dictionary to count characters in string s.
    3. Traverse string t:
    4. Reduce frequency for each character.
    5. If character not present ‚Üí return False.
    6. If all counts become zero ‚Üí return True.

    üíª Solution (Python)

    def is_anagram(s, t):
    if len(s) != len(t):
        return False

    count = {}

    for char in s:
        count[char] = count.get(char, 0) + 1

    for char in t:
        if char not in count:
            return False
        count[char] -= 1
        if count[char] < 0:
            return False

    return True

    ‚è± Time & Space Complexity

    Time Complexity: O(n) : Each string is traversed once. Space Complexity: O(1) : At most 26 characters stored (for lowercase English letters).

    ‚úÖ Problem 2: First Non-Repeating Character

    üìå Problem Statement

    Given a string s, return the index of the first non-repeating character. If none exists, return -1.

    üí° Approach

    1. Count frequency of each character.
    2. Traverse string again.
    3. Return index of first character with frequency 1.

    üß† Detailed Explanation

    1. Use a dictionary to store frequency.
    2. First pass ‚Üí count characters.
    3. Second pass ‚Üí find first with count 1.
    4. If none found ‚Üí return -1.

    This avoids nested loops and improves efficiency.

    üíª Solution (Python)

    def first_unique_char(s):
    freq = {}

    for char in s:
        freq[char] = freq.get(char, 0) + 1

    for index, char in enumerate(s):
        if freq[char] == 1:
            return index

    return -1

    ‚è± Time & Space Complexity

    Time Complexity: O(n) : Two passes through string. Space Complexity: O(1) :Fixed character set size.
